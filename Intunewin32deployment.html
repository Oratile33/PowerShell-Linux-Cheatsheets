Intune Fundamentals · Win32 App Deployment Overview

1. What is Win32 App?
●Win32 apps are traditional Windows applications (EXE, MSI, MSP, BAT, CMD, PowerShell, etc.) packaged and deployed through Microsoft Intune.
●These apps are installed on Windows devices managed by Intune.

3. Win32 App Package Creation
●Use Microsoft Intune Win32 Content Prep Tool to package the application.
●The tool converts the folder into .intunewin file.
●This file is uploaded to Intune.
Using Intune Win32 Content Prep Tool
1
Download IntuneWinAppUtil.exe
2
Create a folder and place all app files.
3
Open CMD in that folder.
4
Run the command:
IntuneWinAppUtil.exe -c "C:\AppFolder" -s Setup.exe
  -o "C:\OutputFolder"
Output: .intunewin file
4. Create Win32 App in Intune
1
Go to: Apps > All Apps > Add
2
Select: Windows app (Win32)
3
Upload the .intunewin file
4
Enter App Information:

5. Detection Rules
●Detection rules help Intune identify if the app is already installed.
●If detected, the app will not be reinstalled.
Rule TypeExample
File File or folder exists
C:\Program Files\App\app.exe
File File version
app.exe version >= 1.0.0.0
Registry Registry key exists
HKLM\SOFTWARE\Company\App
MSI Product Code
{12345678-1234-1234-1234-123456789AB}
