1. What is Azure AD (Entra ID)?
●Azure AD (now Microsoft Entra ID) is Microsoft's cloud identity and access management service.
●It helps organizations manage users, groups, applications, and devices in the cloud.
●Provides Single Sign-On (SSO) to cloud and on-premises applications.
●Secures access with Conditional Access and Multi-Factor Authentication (MFA).

4. Types of Identities
●Member User — Created and managed in Entra ID.
●Guest User — External users invited for collaboration.
●External Identities — Customers or partners using CIAM.
●Managed Identity — Used by applications and Azure services.
